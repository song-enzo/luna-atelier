# 服装订单管理系统 - 设计文档

## 概述
服装生产工厂内部使用的订单管理系统。对外接单，支持客人登录选款、上传花版图、下订单、实时查看进度。

## 技术栈
- **后端**: Python Flask
- **数据库**: SQLite（初期简单，后期可迁MySQL）
- **前端**: 原生 HTML + CSS + JavaScript
- **部署**: NAS Docker 内运行

## 第一期功能（网站）

### 1. 登录系统
- 管理员手动创建客人账号
- 客人凭账号密码登录
- 不同账号只能看到自己的订单

### 2. 款式管理（后台）
- 添加/编辑款式
- 款式信息：名称、编号、参考图、关联布料成分
- 布料成分：如"涤纶65% + 棉35%"

### 3. 花版图管理（后台+客人端）
- 客人上传花版图（整张色卡图或单色图）
- 支持在图片上框选颜色区域（Canvas裁剪）
- 每框一个区域 → 生成一行：色块缩略图 + 数量输入框
- 客人上传过的花版自动保存到图库，以后可选

### 4. 下单流程
```
客人选款式 → 上传花版图 → 框选颜色区域 → 填数量 → 提交
```
- 一个订单 = 一个款式 + 多个花色（每个花色有缩略图+数量）
- 订单生成后状态：待确认

### 5. 订单管理（后台）
- 订单列表（所有订单）
- 订单详情（款式、花色清单、数量、进度、金额）
- 订单状态流程：
  ```
  待确认 → 生产中 → 裁剪中 → 车缝中 → 质检 → 已发货
  ```
- 管理员手动更新进度
- 管理员录入应收/已收金额

### 6. 订单查询
- 按订单号搜索
- 按客人筛选
- 按状态筛选
- 按日期范围
- 历史订单完整保留（含花版图、颜色缩略图）

### 7. 客人端
- 登录后看到自己的订单列表
- 每个订单可查看进度
- 可看到管理员更新的金额信息

## 数据结构

### 表1: users（用户/客人）
| 字段 | 类型 | 说明 |
|------|------|------|
| id | int | 主键 |
| username | text | 登录名 |
| password | text | 密码（哈希） |
| nickname | text | 显示名称 |
| role | text | admin / guest |
| created_at | datetime | |

### 表2: styles（款式）
| 字段 | 类型 | 说明 |
|------|------|------|
| id | int | 主键 |
| name | text | 款式名称 |
| code | text | 款式编号 |
| fabric_info | text | 布料成分 |
| image_path | text | 款式参考图 |

### 表3: orders（订单）
| 字段 | 类型 | 说明 |
|------|------|------|
| id | int | 主键 |
| order_no | text | 订单号（ORD-年月日-序号） |
| user_id | int | 客人ID |
| style_id | int | 款式ID |
| status | text | 状态 |
| total_qty | int | 总数量 |
| total_amount | float | 总金额 |
| paid_amount | float | 已付金额 |
| created_at | datetime | |
| updated_at | datetime | |

### 表4: order_items（订单明细-花色）
| 字段 | 类型 | 说明 |
|------|------|------|
| id | int | 主键 |
| order_id | int | 订单ID |
| color_name | text | 颜色名称（可选） |
| swatch_image | text | 框选的色块缩略图路径 |
| quantity | int | 数量 |
| sort_order | int | 排序 |

### 表5: fabric_images（花版图库）
| 字段 | 类型 | 说明 |
|------|------|------|
| id | int | 主键 |
| user_id | int | 上传者 |
| image_path | text | 花版大图路径 |
| created_at | datetime | |

## 页面清单

### 客人端
1. 登录页
2. 首页（款式列表）
3. 下单页（选款→上传花版→框选颜色→填数量→提交）
4. 我的订单列表
5. 订单详情（含进度）

### 管理后台
1. 登录页
2. 仪表盘（订单概览）
3. 订单管理列表
4. 订单详情（可更新进度、金额）
5. 款式管理（添加/编辑）
6. 客人管理（添加账号）
7. 花版图库

## 部署
- 运行在NAS Docker容器内（现有Hermes容器）
- 端口：5000（Flask默认）
- NAS地址：192.168.1.22:5000
- 局域网内设备（含Win7）均可访问
